# Internal Phase Docs

The `internal/` directory contains the M1 phase contracts used by Telltale agents.

Some similarly named files are intentional aliases kept for prompt readability and historical command wording:

| Alias | Canonical contract |
|---|---|
| `rechart.md` | `re-chart.md` |
| `island-select.md` | `island-selection.md` |
| `close-island.md` | `island-close.md` |

Alias files should remain short pointers. Update the canonical contract file when changing behavior.
